# Hi, I'm Goodwin 👋

Software Engineer · Co-Founder & CEO at [**Crucibel Technology Limited**](https://crucibel.org)

## What I'm Building

### 🏥 Nova - Healthcare Infrastructure for Kenya

Nova is a modern, interoperable healthcare platform focused on:

- Electronic Health Records (EHR)
- Health Information Exchange (HIE)
- Healthcare interoperability
- Pharmacy Management
- Laboratory Information Management
- Diagnostic Imaging
- Patient Portal
- Provider Portal
- Secure clinical data exchange
  | Module | Description |
  |---|---|
  | **EHR** | Electronic Health Records for clinical workflows |
  | **HIE** | Health Information Exchange across facilities |
  | **Pharmacy** | End-to-end pharmacy management |
  | **Laboratory** | Laboratory Information Management System |
  | **Diagnostics** | Diagnostic imaging and reporting |
  | **Patient Portal** | Patient-facing access to records and care |
  | **Provider Portal** | Clinician and facility management tools |
  | **Interoperability** | Secure clinical data exchange across systems |
