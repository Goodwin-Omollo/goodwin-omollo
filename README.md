## What I'm Building

### 🏥 Nova — Healthcare Infrastructure for Kenya

Nova is a modern, interoperable healthcare platform that treats each facility type and workflow as a first-class citizen. Rather than a monolithic system, Nova is designed as a modular ecosystem where each component can operate independently and integrate seamlessly with the rest.

**Core capabilities:**

| Module               | Description                                                 |
| -------------------- | ----------------------------------------------------------- |
| **EHR**              | Electronic Health Records for end-to-end clinical workflows |
| **HIE**              | Health Information Exchange across facilities and regions   |
| **Pharmacy**         | End-to-end pharmacy and dispensing management               |
| **Laboratory**       | Laboratory Information Management System (LIMS)             |
| **Diagnostics**      | Diagnostic imaging, reporting, and PACS integration         |
| **Patient Portal**   | Patient-facing access to records, results, and care history |
| **Provider Portal**  | Clinician and facility management tools                     |
| **Interoperability** | Standards-based data exchange (HL7 FHIR, SMART on FHIR)     |
